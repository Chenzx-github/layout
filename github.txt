非第一次上传：
1、先删除远程 Git 仓库
git remote rm origin

2、如果执行 git remote rm origin 报错的话，我们可以手动修改gitconfig文件的内容
$ vi .git/config
把 [remote “origin”] 那一行删掉就好了
（ 或者找到你的github的安装路径，我的是C:\Users\ASUS\AppData\Local\GitHub\PortableGit_ca477551eeb4aea0e4ae9fcd3358bd96720bb5c8\etc，找到一个名为gitconfig的文件，打开它把里面的[remote "origin"]那一行删掉就好了！）

3、再添加远程 Git 仓库
git remote add origin (建好新库的SSH)
